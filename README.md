# nestjs-boilerplate

## Description
This is try to build a NestJS boilerplate with all frequent needed features. At the same time this is my first project in NestJS.
__Development in progress, currently this is very early version.__

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
